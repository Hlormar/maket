- Врачи должны иметь возможность авторизоваться через МИС, запускать приложение для начала работы с приемом пациента.
- Врачи должны иметь возможность инициировать аудиозапись диалога с пациентом кнопкой "Начало аудиозаписи", при этом автоматически загружая данные о приеме и пациенте из МИС, с последующим ожиданием команды завершения для обработки.
- Врачи должны иметь возможность завершить запись кнопкой "Конец аудиозаписи", после чего система автоматически выполняет транскрибацию, диаризацию участников, выделение ключевых медицинских терминов и преобразование в структурированный медицинский отчёт, отображаемый в окне предпросмотра.
- Врачи должны иметь возможность просматривать сгенерированный отчет и вносить изменения, с автоматическим сохранением всех правок.
- Врачи должны иметь возможность утвердить и отправить финальный отчет в МИС кнопкой "Отправить отчёт в МИС", обеспечивая его сохранение в карточке приема пациента с обновлением статуса.

Авторизация врача

 Описание процесса

- Врач авторизуется в МИС
- Врач запускает приложение
- Врач нажимает на кнопку авторизации в интерфейсе приложения

Результат

- Приложение получает данные о враче из МИС

Начало записи (главное окно)

 Описание процесса

- Врач нажимает на кнопку “Начало аудиозаписи” в интерфейсе
- Программа загружает данные о приеме пациента из МИС

Результат

- Программа ожидает команды “Конец записи”, чтобы начать обработку аудиозаписи

Конец записи (окно записи)

 Описание процесса

- Врач нажимает на кнопку “Конец аудиозаписи” в интерфейсе
- Программа начинает транскрибацию аудиозаписи
- Программа выделяет участников диалога, ключевые слова
- Программа преобразует аудиозапись в текстовую форму, с отмеченными участниками диалога
- Текстовая форма диалога преобразуется в медицинский отчет о приеме пациента

Результат

- Отчет отображается в окне предпросмотра данных для проверки врачом

Редактирование отчета

 Описание процесса

- При просмотре созданного отчета врач может вносить изменения в отчет перед отправкой

Результат

- Любые внесенные изменения сохраняются

Отправка отчета в МИС

 Описание процесса

- Врач нажимает на кнопку “Отправить отчет в МИС” в интерфейсе
- Программа отправляет отчет в МИС в соответствующую карточку приема пациента

Результат

- Отчет отправлен в МИС
